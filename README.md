## Oraculum - Telegram Bot

Робот выводит прогноз на день или на час по китайскому календарю. 
Прогноз по тибетским праздникам, рекомендуемое время стрижки и поездок. 
А так же можно посмотреть лунный прогноз на день и символ дня.

### Команды используемые в Боте

- Главное меню  **/main** 
- Китайский прогноз на день  **/day** 
- Китайский прогноз на Час  **/hour** 
- Тибетский прогноз на день  **/tibet** 
- Лунный прогноз на день  **/moon** 
- Cимволы дня по звёздам **/stars** 
- Профиль пользователя  **/profile** 
- Проверка времени и текущей локации  **/time**
- Для того чтобы посмотреть прогноз на нужную вам дату,
её необходимо ввести в чате бота в формате DD-MM-YYYY (Например: 17-03-2023).
