## Oraculum – Telegram Bot

The bot provides a daily or hourly forecast based on the Chinese calendar.
It also offers forecasts for Tibetan holidays, recommended times for haircuts and travel,
as well as the lunar forecast and the symbol of the day.

### Commands used in the Bot
**/
- Main menu  **/main** 
- Chinese daily forecast  **/day** 
- Chinese hourly forecast  **/hour** 
- Tibetan daily forecast  **/tibet** 
- Lunar daily forecast  **/moon** 
- Day symbols according to the stars **/stars** 
- User profile  **/profile** 
- Time and current location check  time**
- To view a forecast for a specific date,
enter it in the bot chat in the format DD-MM-YYYY (for example: 17-03-2023).
